# CityZen

CityZen is a cross-platform React application that targets web and native mobile using a shared JavaScript codebase. The project uses React Native + React Native Web for UI portability and supports running via Expo (recommended) or a Vite web workflow.

Quick links
- `App.jsx` — main navigation and app entry
- `index.jsx` / `index.html` — web entry
- `src/components/ui/` — shared UI components (many currently web-first)
- `SETUP.md` — step-by-step setup
- `SETUP_COMPLETE.md` — summary of changes performed

Want to get running quickly? See `SETUP.md` for precise instructions for web and Expo.
